== Description ==

O plugin realiza integração de listagem de produtos, informações de eventos, informações de produtos. 

== Frequently Asked Questions ==

= O que é o Retrilhar? =

Retrilhar oferece um conjunto de soluções para empresas que trabalham com turismo de natureza, incluindo e-commerce, pagamento online, controle de reservas, contratação de seguro automaticamente, gestão de cobranças e pagamentos, clubes de assinatura, integração com Google contatos e um fluxo completo de reservas e de atividades que automatizam e auxiliam o trabalho de quem fornece serviços de turismo de natureza.

= Como realizar a integração? =

Basta baixa o plugin e acessar a página de configuração do Retrilhar
